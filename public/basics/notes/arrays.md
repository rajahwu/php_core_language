# Arrays

A collection of data that acts as key-value pairs.

Arrays are great for complex data sets.

## Types

* Indexable arrays

    * numbered index starting at 0

```php
    Colors [
        0 => red
        1 => blue
        2 => green
    ]

    // Size 3 array!
```

* Associative arrays

    * String index

```php
    Home Towns [
        'Joe' => 'Middletown, NY'
        'Erin' => 'West Chester, PA'
        'Dave' => 'Exton, PA'
        'Brian' => 'Grand Rapids, MI'
    ]
```
